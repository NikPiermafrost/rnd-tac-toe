About this project

This project was born because of pure Boredom.

One Saturday i woke up and thought, what about Tic Tac Toe, but cursed?

In just a couple of hours the game was born. Although this doesn't deserve to exist, I created this project to put a smile on your faces (or smash your phone/keyboard/whatever).

The game was made entirely in c# (.net 5), with a asp.net core backend and Blazor Webassembly

Due to the effort in develop it due to poor support for the template linter, and due to the plan to do something bigger and useless, I have scraper Blazor WebAssembly in favour of dotnet 6 minimal api for backend and Angular 13 for the frontend.

I don't really need help to maintain this abomination, but in case you want to contribute in some ways, the project is entirely Open Source.
